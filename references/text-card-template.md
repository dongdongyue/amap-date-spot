## 推荐理由写作要点

- 包含至少一项：距离便利性 / 氛围描述 / 特色亮点
- 一句话，不超过 30 字
- 语气自然，不像广告文案
- 特殊场景融入对应话术（生日→"可以准备小惊喜"，表白→"安静适合说重要的话"）
- 好例子："湖边散步最自然，比面对面坐着压力小"
- 好例子："清吧氛围好，灯光音乐恰到好处"
- 坏例子："这是一家非常优秀的知名咖啡馆" ← 太官方

## 出片建议写作要点

- 基于常识推断，不编造不存在的角落
- 格式：具体位置 + 时间/光线 + 效果
- 不确定时写"环境好，自然出片"
- 好例子："靠窗第三个位置，下午 2-4 点侧光最好看"
- 好例子："门口绿植墙是打卡点，晴天拍更好"
- 坏例子："每个角落都出片" ← 太模糊

## 动线编排写作要点

- 站与站标注步行分钟数
- 每站标注建议停留时间
- 总时长控制在 2.5-4 小时
- 超过 800m 不做强制串联，改为自由切换模式

## 导航链接格式

```
https://uri.amap.com/marker?position=LNG,LAT&name=URL_ENCODED_NAME
```

- position 用 POI 的 location 字段（lng,lat）
- name 用 POI 名称做 URL 编码（中文需 encodeURIComponent）
